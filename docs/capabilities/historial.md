# Capacidad: Historial

## Objetivo

Visualizar la línea de tiempo de eventos asociados a un envío.

## Caminos felices

| Escenario | Estado |
|---|---|
| Ver historial de envío con eventos | Pendiente |
| Ver detalle de evento | Pendiente |

## Caminos excepcionales

| Escenario | Estado |
|---|---|
| Historial vacío | Pendiente |
| Envío sin eventos | Pendiente |
| Error al cargar historial | Pendiente |

## Estado E2E

Pendiente.

## Dependencias

- Envío existente.
- Guía válida.
- Eventos semilla.
- Resultado observable en UI.

## Próxima acción

Automatizar después de tener creación o datos semilla de envío.
