# Capacidad: Envíos

## Objetivo

Gestionar la creación y seguimiento inicial de envíos logísticos.

## Caminos felices

| Escenario | Estado |
|---|---|
| Crear envío con datos válidos | Pendiente |
| Generar guía de rastreo | Pendiente |
| Registrar primer evento de seguimiento | Pendiente |

## Caminos excepcionales

| Escenario | Estado |
|---|---|
| Crear envío con datos obligatorios vacíos | Pendiente |
| Crear envío con dirección inválida | Pendiente |
| Crear envío sin destinatario | Pendiente |
| Crear envío sin origen o destino | Pendiente |

## Estado E2E

Pendiente.

## Dependencias

- Login estable.
- Rol operador o administrador.
- Formulario disponible.
- Datos controlables.
- Resultado observable.
