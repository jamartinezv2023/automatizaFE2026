# Envíos semilla

## Objetivo

Definir envíos base para pruebas de historial, rastreo y notificaciones.

## Envíos requeridos

| Envío | Estado | Uso | Estado dataset |
|---|---|---|---|
| Envío creado | Creado | Validar creación | Pendiente |
| Envío en tránsito | En tránsito | Validar rastreo | Pendiente |
| Envío entregado | Entregado | Validar cierre | Pendiente |
| Envío con incidencias | Retenido/Reprogramado | Validar excepción | Pendiente |

## Dependencias

- Rol operador.
- API o UI de creación.
- Base de datos controlable.
- Limpieza de datos posterior.
