# Plan de aseguramiento de calidad IEEE 730

## Propósito
Definir prácticas mínimas de aseguramiento de calidad para el proyecto de automatización.

## Alcance
Automatización E2E funcional del sistema LogiTrack.

## Estándares aplicados
- Serenity BDD
- Screenplay Pattern
- Cucumber BDD
- Principios FIRST
- Patrón AAA
- Evidencia trazable

## Gestión de defectos
Todo comportamiento no automatizable de forma estable debe documentarse como hallazgo.

## Evidencia
Los reportes Serenity y los documentos en docs/hallazgos soportan la trazabilidad.
