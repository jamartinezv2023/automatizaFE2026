# Roadmap E2E

## v1.7.x - Cobertura medible

- Crear dashboard de cobertura funcional.
- Crear backlog de automatización.
- Clasificar riesgos.
- Priorizar siguientes escenarios.

## v1.8.x - Rastreo funcional

- Confirmar contrato de rastreo por guía.
- Identificar guía válida o respuesta esperada.
- Automatizar consulta si existe resultado observable.

## v1.9.x - Registro excepcional

- Validar campos obligatorios.
- Validar correo inválido.
- Validar contraseña inválida si la UI lo permite.

## v2.0.x - Autenticación

- Diagnosticar login con evidencia HTTP.
- Definir si el bloqueo es frontend, backend, token o cookie.
- Automatizar login solo si el flujo se estabiliza.

## v2.1.x - Dominio logístico

- Explorar creación de envíos.
- Explorar historial.
- Explorar notificaciones.
- Automatizar únicamente funcionalidades testeables bajo INVEST.
