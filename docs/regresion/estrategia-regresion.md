# Estrategia de regresión basada en riesgo

El objetivo no es automatizar todo, sino ejecutar primero aquello cuyo fallo genera mayor impacto sobre el negocio.

## Suites

@smoke
@critical
@happy
@exception
@regression
@discovery
@blocked

## Orden de ejecución

1. Smoke
2. Critical
3. Happy
4. Exception
5. Regression
6. Discovery
